# Tracker Backend

This is a simple backend service for tracking items.

## Installation

1. Install dependencies:
   ```bash
   pip install fastapi pydantic uvicorn
   ```

## Usage

1. Run the server:
   ```bash
   uvicorn main:app --reload
   ```

2. Access the API documentation at `http://localhost:8000/docs`.
